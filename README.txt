== Description ==
 
This plugin creates a custom post type: Book and 2 taxonomies: Authors & Publishers, as well as allows users to search books from the search page. It also registers a shortcode: [library_book_search]. You can use this shortcode to render the search template anywhere throughout your wordpress site.
 
== Installation ==
 
1. Upload the entire library-book-search folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
